Add, edit and delete WP and plugin options. You can edit serialized data as well.

== Description ==

WARNING! THIS PLUGIN IS FOR ADVANCED USERS AND DEVELOPERS ONLY!

This plugin allows you to perform the following tasks from the WordPress dashboard.

* Edit serialized options directly or as a JSON string.
* Create new options
* Delete options

== Installation ==

1. Just like you would install any other plugin.
